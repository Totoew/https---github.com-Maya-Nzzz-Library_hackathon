Подробное описание зон

5 Абонемент для детей

Специализированная зона для юных читателей с подборкой детской литературы, игровой зоной и пространством для творческих занятий.

7 Гардероб

Зона для хранения верхней одежды посетителей, оборудованная современными системами хранения.

9 Абонемент

Основная зона выдачи литературы для взрослых читателей, оснащенная автоматизированной системой учета книг.

10 Событийный зал

Многофункциональный зал, оборудованный современной мультимедийной техникой для проведения лекций, презентаций, мастер-классов и других мероприятий.

11 Хранение книг

Специализированное помещение для хранения книжного фонда с регулируемым микроклиматом и системой безопасности.

12 Молодежный зал

Многофункциональный зал с амфитеатром, предназначенный для мероприятий, встреч и работы молодежных сообществ.

13 Зал искусств

Арт-галерея и многофункциональное пространство для выставок, перформансов и творческих встреч.

13.1 Проектный офис

Рабочее пространство для сотрудников библиотеки, занимающихся разработкой и реализацией проектов.

13.2 Медиасреда (подкастерная)

Современная студия для записи подкастов, оснащенная профессиональным звуковым оборудованием.
